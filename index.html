An instant message application

An instant message application(chat) with Meteor.js

Project Description

In this web application the point is to chatting. Basically, the way it works is that you log in as some user and choose someone to chat with. After you've finished, log in as the user you chose in other browser and open chat with the user logged in in the previous browser. Then type a message. The key thing is that when you send a message you see it immediately in both browsers.

Features realized in this web application: templates, emoticons, publish and subscribe model to control data read access, user accounts package to create a log in form, iron:router package to create multiple page applications, automatic data entry forms generation using Simple Schema and autoform.

Skills used to create the project:

  • HTML
  • CSS
  • Bootstrap
  • JavaScript
  • JQuery
  • Meteor
  • MongoDB
  • Responsive Web Design
